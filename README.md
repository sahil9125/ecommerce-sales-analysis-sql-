# E-Commerce Sales Analysis using PostgreSQL

## Project Overview

This project demonstrates SQL-based sales analysis for an e-commerce company using PostgreSQL. The database includes customers, products, orders, and order items data.

## Objectives

* Analyze customer spending
* Identify highest sales generating city
* Find category-wise sales performance
* Track product sales quantity
* Generate business insights using SQL queries

## Tools & Technologies

* PostgreSQL
* SQL
* Joins
* Aggregate Functions
* Group By
* Having Clause

## Database Tables

* customers
* products
* orders
* order_items

## Key SQL Concepts Used

* INNER JOIN
* GROUP BY
* HAVING
* SUM()
* AVG()
* COUNT()
* ORDER BY
* LIMIT

## Business Questions Solved

1. Total orders and spending by each customer
2. Highest sales generating city
3. Average price and sales by product category
4. Customers spending more than ₹50000
5. Order summary of last 10 months
6. Most sold product by quantity
